# Projects Description
This folder contains my complete collection of Machine Learning projects, organized from fundamentals to advanced levels. It includes hands-on implementations of essential ML algorithms, real-world datasets, deep learning models, and practical problem-solving approaches.

## Technology used
- Python
- SQL
- Scikit-learn, OpenCV , PyTorch
- Django , NumPy & Pandas
- HTMX
- Bootstrap
- Matplotlib / Seaborn etc

## To set up this project

1. cd into project directory
2. Create your environment "  python -m venv venv_name "
3. Install dependencies through command `pip install -r requirements.txt`
4. Run the main.py files
